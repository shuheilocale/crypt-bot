## Description
This script is cryptocurrency portfolio viewer on slackbot.

## Requirement
You need these libs.
* slackbot
* pycurl
* urllib
* requests
* hmac
* hashlib
* pybitflyer
* zaifapi
* coincheck
* matplotlib
* numpy

## Install
1. Rename _amount_sample_.json to _amount.json_
2. Update amounts you have.
3. Rename _slackbot_settings_sample.py_ to _slackbot_settings.py_
4. Update keys.
